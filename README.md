##This project is an end-to-end Machine Learning application that predicts student math scores based on demographic and educational attributes such as gender, parental education, lunch type, test preparation course, reading score, and writing score.

#The project follows a complete production-level ML pipeline including:

Data Ingestion
Data Transformation
Model Training
Prediction Pipeline
Flask Web Application
Model Serialization
Deployment Ready Structure

##The application is built using Python and follows modular software engineering practices for scalability and maintainability.

Tech Stack
Python
Scikit-learn
CatBoost
Pandas & NumPy
Flask
HTML/CSS
Pickle
GitHub Actions

##Features

Predicts student performance accurately
Complete training and prediction pipelines
Web interface using Flask
Modular project structure
Deployment-ready architecture
Automated workflow support with GitHub Actions
